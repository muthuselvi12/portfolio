Project

Abstract

This project presents an AI-powered safety system for railway transportation that automatically detects obstacles and animal crossings on railway tracks and enables self-stopping of trains to prevent accidents. The system integrates a LiDAR sensor with a machine learning model to monitor the tracks in real-time. When an obstacle—such as a rock, vehicle, or animal—is detected within a dangerous proximity, the system processes the spatial data and sends an automatic stop signal to the train engine control unit. The goal of the project is to reduce railway accidents caused by unexpected obstructions and to enhance autonomous decision-making in rail systems. The system also stores detection events and can alert operators for logging and emergency response.

Technologies & Tools Used:

  • Sensors: LiDAR (Light Detection and Ranging).
  • AI/ML: Object detection/classification.
  • Hardware Interface: Arduino.
